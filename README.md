# TH3V151T0R5_B 전투봇

마스레이드 PVP 팀전 자동 정산 봇.
구글 시트 커맨드 입력 → 전투 계산 → 마스토돈 퍼블릭 툿 출력.

---

## 구조

```
main.rb               진입점, 30초마다 실행 시트 폴링
sheet_manager.rb      구글 시트 읽기/쓰기
battle_calculator.rb  명중/회피/크리티컬/피해 계산
battle_processor.rb   지원→이동→공격→방어 순 정산
toot_builder.rb       결과 텍스트 생성 (500자 단위 스레드)
mastodon_client.rb    마스토돈 퍼블릭 툿 전송
```

---

## 시트 구조

### 운영 시트 (OPS_SHEET_ID)

| 탭 | 용도 |
|----|------|
| 실행 | A2=ON/OFF 체크박스, B2=라운드, C2=턴(1=선공/2=후공) |
| 보정 | A=캐릭터명, B=항목, C=값, D=적용여부, E=메모 |
| 스탯 | A=캐릭터명, B=건강, C=마법능력, D=내구도, E=민첩, F=기술, G=행운, H=스킬1, I=스킬2 (2행부터) |
| 스킬 | A=스킬명, B=분류, C=사거리, D=쿨타임, E=설명 (2행부터) |

### 러너 커맨드 시트 (RUNNER_SHEET_ID)

| 탭 | 용도 |
|----|------|
| 현황 | A=캐릭터명, B=현재건강, C=현재위치 (봇이 자동 업데이트) |
| 커맨드 | A=캐릭터명, B=이동위치, C=행동, D~H=대상1~5 |

## 보정 항목

| 항목 | 값 | 설명 |
|------|----|------|
| 체력보정 | 숫자 (음수 가능) | 수동 체력 수정 |
| 사거리불가 | - | 해당 캐릭터 행동 무효 |
| 사망 | - | 강제 사망 처리 |
| 부활 | 좌표 (예: C3) | 해당 위치에 최대 체력으로 부활 |
| 쿨타임초기화 | - | 수동 처리 메모용 |

## 계산식

```
명중률    = 60 + 기술 (%)
회피율    = 민첩 × 2 (%, 자동 판정)
크리티컬  = 행운 × 2 (%, 자동 판정)
피격 대미지 = (타격 대미지) - (내구도)
크리티컬 성공 시 마법 능력 2배로 계산
```

## 운영 순서

1. 러너들이 커맨드 탭에 이동위치·행동·대상 입력
2. 보정 필요 시 보정 탭에 입력 후 D열 체크
3. 실행 탭 B2=라운드, C2=턴 입력
4. 실행 탭 A2 체크박스 체크
5. 봇이 자동으로 계산 → 현황 탭 업데이트 → 마스토돈 툿 전송
